# Pigment Room
## 初めに

これは[PigmentRoom](https://pigment.ykvario.com/ "PigmentRoom")の基礎ソースコードです。  
カラーコードの調査や配色、UI周りの調整などにご利用ください。  

広告以外では外部通信は存在しませんが、ローカルで使用したい場合はこちらをクローンしてご利用ください。  

Web屋がWeb屋（自分）のために作ったものなので、機能を追求しました。

## 使い方

### 色選択 / 変換
色選択 / 変換では、色を選び、各種カラーコードに変換することができます。  
右側の四角でパレットが出てきます。  
パレットではRGPの入力、HEXの入力も可能です。それ以外は、入力はできないので、適当なイラレなどを使用し、色をスポイトでとってください。  

### 組み合わせ
組み合わせでは、最大8色まで色の組み合わせを見ることができます。  
ここで登録している色は左右のボタンを使用することで、配置を移動することができます。  
また、サイトのカラースタイルをCustomにした場合、組み合わせに登録されている色で構成されます。  

### カラースタイル
カラースタイルは4種類です。
| style | image |
| ----- | ----- |
| Light | 一般的な白（#FAF9FF）を基調としたものです |
| Gray | グレー（#EAECF4）を基調としたものです。 |
| Dark | 黒（#121729）を基調としたものです。 |
| Custom | 現在のUIに対して、組み合わせに登録されている色を使って配色します。 |

### セット
配色のプリセットです。  
ローカルでは、sets.jsonで登録することができます。  
ガチャでは、全体からランダムに表示できるほか、  
各種タグをつけておくことで、絞り込みが可能です。  
追加してほしい配色セットがあれば、タグとカラーコード（3種以上）を合わせてご連絡いただければ幸いです。  
[サイト](https://pigment.ykvario.com)に追加させていただきます。

### ローカル保存
登録した組み合わせを保存することができます。  
完全ローカルでの保存なので、どこかに送られることはありません。（このサイトの保存は、すべてローカル端末で完結しています。）

### プリセット
いわゆるカラーパレットです。

### 詳細保存
手打ちによって、保存することができます。  
無色を追加することができ、使用すると、現在使用しているカラーが存在する場合に限り、無色は現在の色を使用、それ以外は置き換える。という挙動になります。  
例：#ECEFF4,#2E3440,#88C0D0,xxxxxx,#2563EB,xxxx,#F08991  
4色目と6色目が、現在使用している色を保持します。  
説明テキストの保存も可能です。カードにホバーすることで、保存した説明文を表示させることができます。
